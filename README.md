# Healthy Mom

A pregnancy companion app providing week-by-week guidance, health information, and community support for expectant mothers.

## Features

- Week-by-week pregnancy tracking
- Health library with articles
- Clinic finder
- Community forum
- Mood tracker
- Kick counter
- Contraction timer
- Appointment management
- Daily reminders
- Nutrition guidance
- Bilingual support (English/Swahili)

## Tech Stack

- React + Vite
- Node.js + Express
- MongoDB
- JWT Authentication
